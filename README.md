# Express Portfolio

Portfolio website yang dibangun dengan Express.js dan Handlebars template engine.

## Deskripsi

Aplikasi ini adalah sebuah website portfolio sederhana yang menampilkan informasi personal, skill, proyek dan kontak.

## Fitur

-  Section Home
-  Section Skills
-  Section Projects
-  Section Experiences
-  Section Contact

## Tech Stack

- **Backend**: Express.js
- **Template Engine**: Express Handlebars (HBS)
- **Frontend**: HTML, CSS, JavaScript, Bootstrap

## Instalasi

1. Clone atau download project ini
2. Install dependencies:
   ```bash
   npm install
   ```

## Menjalankan Aplikasi

Jalankan server dengan perintah:

```bash
npm run start
```

Server akan berjalan di `http://localhost:3000`
